## 📦 Composant

Ce composant vue-loader pour Vue-Cli permet de partager vos réseaux sociaux à travers votre application

## 📝 Installation

```
git clone https://github.com/padcmoi/vue-social-sharing
npm install
```

### 🔨 Développement mode

```
npm run serve
```

### 🔨 Build

```
npm run build
```
